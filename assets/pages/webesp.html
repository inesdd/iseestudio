PROYECTOS WEB

ISEE estudio, lleva a cabo todo tipo de proyectos sobre páginas web a partir de HTML, animación CSS, JavaScript y Bootsrap. En este caso se expone la página web llevada a cabo por un proyecto sobre customización de zapatillas y todo tipo de accesorios relacionados con marcas deportivas.

Las páginas se realizan a partir de un prototipo previo en Figma, para poder trabajar de forma eficiente y creando un diseño trabajado y serio.

Analogically, es un proyecto cuya temática principal es la fotografía analógica, y consistiría en un pequeño blog donde cualquier persona interesada en este ámbito, pueda iniciarse y buscar información, además de servir como medio de contacto con otros usuarios con aficiones similares.

Otro de los proyectos llevados a cabo ha sido la web de meditación y midfulness llamada Aura, donde se pueden encontrar todo tipo de cursos online, tanto para principiantes que quieran iniciarse en este mundo, como para personas experimentadas para seguir disfrutando y creciendo con este estilo de vida.
